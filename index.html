The purpose of this website

This website main purpose is to increase my knowledge of JavaScript, here I practise and try to understand many design patterns of different content panels.

I will provide JavaScript scripts for anyone willing to understand or see how any part of the website works and is created. The main resource where I will generate my ideas from is from a book I read Javascript and jQuery by Jon Duckett, the ideas will be taken from it while I will learn from using the example code and somehow changing it and developing slightly advanced JavaScript.

Accordion

By using event object with simple 7 jQuery lines we can create accordion.

  • Event object.
  • .next, .not, .slideToggle() methods.

The purpose of this content panel is to show information by sliding it down. It works by using event object that checks if the tab content tab has class of .animated and then by using slideToggle function it simply opens and closes.

Tabbed panel

This script creates functional tabbed panel.

  • Event object.
  • If statement.
  • jQuery .removeClass, .addClass, .find, .attr functions.

The way this script works is by hiding other tabs and showing only one. It changes the class of tab and its link when clicked on at the same time deleting the active class for tabs that doesn't need to be shown.

Photo Viewer

This sript creates functional Photo Viewer and it also involves, asynchrononous loading and image caching.

  • JavaScript cache object.
  • .fade, .stop., .eq, and more jQuery functions.
  • Event listeners and objects.
  • Caching images.

The way this script works is by creating a cache object where each picture will be loaded and then kept if user presses on any of thumbnails again, it is done by IF statement that checks whether or not image is in cache object which launches another function that shows the loading GIF while picture is loaded at first.

Photo Slider

This photo slider can slide images from left to right and it also slides images automatically every four seconds.

  • .each ,.length, .append, .css jQuery functions.
  • Array to store buttons.
  • Multiple functions and if statements.

This script works by positioning images next to each other but only shows one at a time. It assigns index to each slide and gives each button a link to each index, the index also changes the way images comes in.
